React Redux

A python HTTP WSGI Server for UNIX. It allows you to run any python application concurrently by running multiple python processes(multi-threaded).
It the most prefered server working with frameworks like Django and Flask etc. in production.

mapStateToProps

A function that simply returns a specified part of the current state from redux to a component.

The "connect" function "binds" mapStateToProps to the Header component

The returned state (currentUser & hidden) from mapStateToProps will be available to be accessed in the Header component

Any changes made in those state in their respective reducer in redux will be reflected and updated in the binded component(Header)
